Network Monitor basic speed test

The Network Monitor speed test is a simple speed test which may not be as accurate as tests performed by other apps which specialize in speed tests.

Network Monitor is a completely free app, with no ads, maintained by one hobbyist, and the author of this app cannot pay for a dedicated server for thousands of users to constantly run speed tests.

So, in Network Monitor, you must choose a server on which to perform the speed test.

Download test (http)

For the download speed test, you select a url of any file to download. The bigger the file, the more accurate the test will be, but the greater the possibility of exceeding your data quota, if you have one.

Upload test (ftp)

For the upload speed test, Network Monitor will upload the file downloaded from the download speed test, to an FTP server. You must have access to an FTP server and provide the login credentials, if you wish to perform the upload speed test.

If you wish, you may run only the download speed test without running the upload speed test.
